What is this?

An easy method to test your own CSS on standard HTML elements. Test some styles for an existing site. Make some boilerplate to kick off a project! Fork or clone this repo for Jekyll on GitHub if you want to try it out yourself with different styles or additional HTML elements or patterns.

Why a Jekyll fork?

Why not? I wanted to implement the a version of Pattern-Primer for my Jekyll projects with only Jekyll dependencies (and no PHP dependencies). It can be built locally with a Jekyll/Ruby environment or uploaded as a static directory on a remote server.
